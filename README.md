# BACK-END Roblox

Projeto em NodeJS + Typescript
Parte do Back-End do Projeto [Roblox](https://b7web.com.br)

## Instalação
**1 -** Criar arquivo .env.
**2 -** Setar Variáveis:
PORT
MONGO_URL
JWT_SECRET_KEY
**3 -** Rodar comandos no Terminal:
```sh
$ npm install
$ npm run start-dev
```
